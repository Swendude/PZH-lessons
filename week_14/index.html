Deze pagina krijgt kleur van css

  • Css staat voor Cascading Style Sheets
  • Css wordt gerendered in de browser (net als HTML)
  • SASS is een css pre-processor
  • Vaak maken frontenders ook gebruik van css frameworks (Tailwind, Bootstrap, ..)
  • HTML tags kunnen ALTIJD 'id' en 'class' attributen hebben
  • CSS kan op drie plekken 'leven'
    • Inline (op het html element)
    • In de head in een 'style' tag
    • Externe .css file (stylesheet), dit is de beste optie
